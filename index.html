2D TOWER DEFENSE GAME

Scratch built 2D tower defense game developed using Unity 2017.4.1f1 and C# for the scripts. No lore to the gameplay, just place your towers and don't let the ladybugs eat the alien's cookie. LOW-POLY GAME

A low-poly game developed using the Java Game Engine, albiet with some hefty editing. Generates a terrain from the specified dimensions with random heights, uses Perline noise to ensure terrain has some smoothness between vertices of the triangle mesh. Water is also rendered in a similar way although uses some mathematics to give the illusion of the water moving
