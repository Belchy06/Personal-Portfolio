My Electronics Projects

SCALE ROCKET FLIGHT COMPUTER

This project started after recieving inspiration from Joe Barnard and his attempt with the BPS.Space project. Using components such as the Wemos D1 Mini, GY-86 10 Degress of Freedom IMU, 12C EEPROM as well as a battery management shield and a SD card shield, the flight computer is able to read values from the IMU and calculate it's position in 3D space and write these values to the EEPROM as a buffer and later the SD card.
